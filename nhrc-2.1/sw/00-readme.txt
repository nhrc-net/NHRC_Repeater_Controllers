This directory contains a COPY of the NHRC-3.1 code.

The NHRC-2.1 is built from the SAME ASM FILE as the NHRC-3.1

Bug fixes and modifications should be made in the NHRC-3.1 folder, 
then copied here after testing.

The nhrc-2_1.mcp project file DOES NOT DEFINE the NHRC3_1 macro that
"turns on" the NHRC-3.1 specific code.
